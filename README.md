Fitur Utama

Website ini memiliki 5 section utama:

1. Hero / Home – Perkenalan dengan foto profil dan tombol call to action.
2. About Me – Informasi lengkap dan identitas diri.
3. Skills – Empat kategori keahlian: IT, UI/UX, Digital Art, dan Traditional Art.
4. Portfolio – Enam karya proyek dengan efek hover.

Cara Menggunakan

1. Menampilkan Website
   Buka file index.html di browser atau gunakan Live Server di VS Code.

2. Mengganti Gambar
   Ganti file placeholder dengan gambar asli Anda:

* profile.png untuk foto profil.
* about.png untuk section About.
* project1-6.png untuk gambar portfolio.

3. Mengedit Konten
   Edit bagian berikut di file index.html:

* Informasi pribadi seperti nama, NPM, dan deskripsi.
* Daftar skills sesuai keahlian Anda.
* Judul dan deskripsi proyek pada portfolio.
* Email, nomor, dan link sosial media pada bagian contact.

4. Kustomisasi Warna
   Ubah variabel warna di file css/style.css pada bagian :root untuk menyesuaikan tema.

Troubleshooting

Jika gambar tidak tampil:

* Pastikan nama file sesuai dan case-sensitive.
* Periksa kembali path folder asset.
* Pastikan file gambar benar-benar ada di folder tersebut.

Jika menu tidak bekerja:

* Pastikan JavaScript aktif di browser.
* Periksa console untuk melihat error.
* Refresh halaman.

Jika style tidak tampil:

* Bersihkan cache browser.
* Periksa path file CSS di HTML.
* Pastikan style.css berada di folder css.
